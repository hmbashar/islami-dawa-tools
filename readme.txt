## Description

Islami Dawa Tools is a professional WordPress plugin designed to provide a complete toolkit for Islamic Dawa (outreach) initiatives. Built with a modern, extensible architecture following WordPress best practices, it includes:

### Key Features

- **Gravity Forms Integration**: Seamless integration with Gravity Forms
- **Multi-Currency Support**: Including BDT (Bangladeshi Taka) currency
- **Professional Structure**: Scalable architecture for easy feature expansion
- **PSR-4 Autoloading**: Clean, organized code structure

### BDT Currency Support

The plugin adds native support for Bangladeshi Taka (BDT) currency in Gravity Forms with proper formatting:
- Symbol: ৳ (Bengali Taka Sign)
- Proper decimal and thousand separators
- Full compatibility with Gravity Forms payment features

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- Gravity Forms (optional, for enhanced form features)

## Installation

1. Upload the plugin to `/wp-content/plugins/` directory
2. Install dependencies: `composer install`
3. Activate the plugin in WordPress admin panel

## Development

This plugin follows WordPress coding standards and uses:

- PSR-4 autoloading via Composer
- WordPress coding standards via PHPCS
- Professional documentation and code comments
